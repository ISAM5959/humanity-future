# Настройка Figma MCP Server

Этот документ содержит инструкции по настройке подключения к Figma MCP серверу.

## URL сервера
```
http://127.0.0.1:3845/mcp
```

## Настройка в Cursor

### Вариант 1: Через настройки Cursor

1. Откройте настройки Cursor (Cmd+, или File → Preferences → Settings)
2. Найдите раздел "MCP Servers" или "Model Context Protocol"
3. Добавьте новый сервер:
   ```json
   {
     "figma": {
       "command": "http://127.0.0.1:3845/mcp",
       "type": "http",
       "transport": "http"
     }
   }
   ```

### Вариант 2: Через конфигурационный файл

Если Cursor использует конфигурационный файл, добавьте в него:

```json
{
  "mcpServers": {
    "figma": {
      "url": "http://127.0.0.1:3845/mcp",
      "transport": "http"
    }
  }
}
```

## Проверка подключения

Убедитесь, что Figma MCP сервер запущен на `http://127.0.0.1:3845/mcp`:

```bash
curl http://127.0.0.1:3845/mcp
```

## Примечания

- Убедитесь, что MCP сервер запущен перед использованием
- Для работы требуется активное подключение к серверу
- Порт 3845 должен быть доступен локально
